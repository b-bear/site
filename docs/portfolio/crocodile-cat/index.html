# Crocodile Cat

Crocodile Cat is a one touch action/stratagy/timing mobile game where you have to escape the jaws of a crocodile and rescue kittens with the use of your trusty jetpack and some explosive pickups. Crocodile Cat is being made with Michael Bowerman (opens new window) in Unity.

Read more about the development of Crocodile cat on Michael's Blog (opens new window).
